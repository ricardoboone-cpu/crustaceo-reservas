\# 🦀 Sistema de Reservas - Crustáceo Cascarudo



\## 📌 Descripción

Aplicación web para realizar reservas en un restaurante.



\## 🏗️ Arquitectura

Usuario → EC2 → Docker

\- Frontend (Nginx)

\- Backend (Node.js)

\- Base de datos (MySQL)



\## ⚙️ Tecnologías

\- Node.js

\- MySQL

\- Docker

\- AWS (EC2, S3, CloudFormation)
